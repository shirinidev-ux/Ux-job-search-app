# Job Portal UX Research & Design

This project focuses on designing a user-friendly job portal based on UX research, targeting freshers and recruiters.

## Problem Statement
Many job portals ignore UX research and fail to address the real needs of job seekers and recruiters.

## Objective
To design a job portal UI using UX research and user-first design principles.

## Tool Used
- Figma (UI/UX Design)

## Project Scope
This is a UX Research and UI Design project. No frontend code implementation is included.

## Screens Designed
- Job Search Screen
- Job Details Screen
- Resume Upload Screen

## UX Research
User research was conducted through informal interviews with students and freshers to identify major pain points.
